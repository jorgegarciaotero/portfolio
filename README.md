# Jorge García Otero's Portfolio

This project is a personal portfolio created using React for the frontend and Node.js for the backend. It showcases information about my education, skills, work experience, and projects.
https://jorgegarciaotero.github.io/portfolio/

## Features

- **Home:** A brief introduction and quick links.
- **Resume:** Detailed information about my education, programming skills, and work history.
- **Projects:** A gallery of my latest projects (*Vodafone Fotodered*, *Vodafone Vision*, *Network Operations KPIs*).
- **Contact:** Form to get in touch with me directly.

## Technologies Used (Portfolio)

- **Frontend:** React, CSS, JavaScript
- **Hosting:** GitHub Pages
- **Assets:** FontAwesome, React Simple Typewriter

## Installation and Setup

### Prerequisites

- Node.js (v14 or higher)
- npm

### Run Locally

```bash
git clone https://github.com/jorgegarciaotero/portfolio.git
cd portfolio/frontend/client
npm install
npm start
```
